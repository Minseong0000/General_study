#Illust
일러스트레이터 단축키, 선그리는법, 편집법

알아둬야 할 단축키
alt + 마우스 휠 : 확대 및 축소
Ctrl + 1 : 100% 크기의 이미지를 화면 정중앙에 셋팅
Shift + 오브젝트의 모서리로 축소 및 확대 : 동일 비율 크기 조정
(Shift +) Ctrl + [ or ] : 화면 (맨)앞, 뒤로 보내기
pen tool + shift : 직선, 수직, 45도 선 잇기


#Git&Github
git의 기본값 설정 및 사용법, 주요 터미널 명령어:파일 생성, 파일 제거
로컬 저장소 만드는법, 사용자 정보등록 

주요 명령어
git add . : 모든 변경사항 커밋 대기 상태로 전환(빨간글씨->초록글씨)
git commit -am "msg": (최초 커밋을 사용한 이후) add & commit 동시 적용
